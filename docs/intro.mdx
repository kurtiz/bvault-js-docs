# bVault-js - Secure Frontend Encryption Library

**bVault-js** is a type-safe, lightweight, zero-dependency cryptographic library for secure encryption and decryption in
browser environments.

It implements **AES-GCM** with **PBKDF2 key derivation**, providing a simple API for password-based encryption of
sensitive data.

:::tip Why use bVault-js?

- Works directly in the **browser** (no native dependencies)
- Strong **AES-GCM 256-bit encryption**
- Built-in **secure wrappers** for localStorage and sessionStorage
:::
